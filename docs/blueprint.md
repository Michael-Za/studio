# **App Name**: صنايعي एक्सبرس (Sanaay3y Express)

## Core Features:

- تسجيل بيانات العميل: العميل يسجل بياناته (اسم، رقم تليفون، صورة البطاقة) و بيانات حسابه البنكي عشان الدفع.
- تسجيل بيانات الحرفي: الحرفي يسجل بياناته (اسم، رقم تليفون، صورة البطاقة) و بيانات حسابه البنكي عشان الدفع.
- نشر طلب خدمة: العميل ينشر طلب: يحدد نوع الخدمة (سباكة، نجارة، إلخ)، وصف للمشكلة، ميعاد مناسب، والميزانية.
- عرض الطلبات: الحرفيين يشوفوا الطلبات المنشورة، ويقدموا عروضهم للعميل.
- مقارنة العروض: العميل يقارن العروض ويختار الأنسب بناءً على السعر والتقييم.
- تأمين الخصوصية: تأمين بيانات المستخدمين وخصوصيتهم بالكامل.

## Style Guidelines:

- اللون الأساسي: أزرق سماوي فاتح (#5999D3) يوحي بالثقة والاحترافية.
- الخلفية: لون رمادي فاتح جدًا (#F2F2F2) لراحة العين.
- اللون الثانوي: أصفر ذهبي (#FFC72C) لإبراز عناصر معينة زي التقييمات.
- خط العناوين و النصوص: 'Inter' sans-serif علشان يكون واضح وعصري.
- أيقونات بسيطة وواضحة بتعبر عن أنواع الحرف والخدمات المختلفة.
- تصميم بسيط و سهل الاستخدام، بيعتمد على تقسيم واضح للمعلومات.
